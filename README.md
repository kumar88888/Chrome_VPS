#  Chrome for vps

Installing Chromium browser for VPS


## Hardware requirements:

| **Hardware** | **Minimum Requirement** |
|--------------|-------------------------|
| **CPU**      | 4 Cores                 |
| **RAM**      | 4 GB                    | 
| **Disk**     | over 10 gb SSD          |

---

## 1-click Script
```
wget https://raw.githubusercontent.com/kumar88888/Chrome_VPS/refs/heads/main/chrome.sh && chmod +x chrome.sh && ./chrome.sh
```
